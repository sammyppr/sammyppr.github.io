[コラム]腰を痛めないで

8月末にいきなりぎっくり腰...

朝起きて、起きあがろうとしたら...いきなり激痛が走る。海外では「魔女の一撃」というらしい

3日間起きられず...

夏休み中だったので、仕事を休み大人しく静養

だましだまし回復を願う

いろんな体制を試し、立ち上がることに成功。

立ち上がるとどうにかなったので、近所を散歩、駅まで散歩など歩ける距離を伸ばし、なんとか仕事に行く様になる。

仕方なく整体に

一向に痛みが取れないため、整体に行くと、「筋肉硬いですねー。左右のバランス悪いですねー」と言われる。

整体師からレントゲンを勧められる

3回ほど通ったが、まだ痛みが取れないのはおかしいので、レントゲンを撮りに行ってください、と助言を受ける。

レントゲン撮ってみたら...
