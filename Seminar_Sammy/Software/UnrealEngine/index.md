# UnrealEngine

[ゼミ に戻る](../../index.md)

## 概要
UnrealEngineはゲームを作るだけでなく、映像、モーショングラフィックス等様々な用途で活用できます。
起動に時間がかかるのでコンピュータのスペックは高い方がいいです。非力なマシンだとイライラします。
